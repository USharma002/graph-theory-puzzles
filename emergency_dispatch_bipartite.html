Maximum Bipartite Matching

Problem Statement

A Bipartite Graph is a network divided into two distinct sets (e.g., Helicopters and Hikers) where connections only go from one set to the other, never within the same set.

A Matching is a subset of edges where no two edges share the same node. The goal is to find a matching with the largest possible number of edges.

Real World Applications

This exact algorithm (often solved via the Hopcroft-Karp algorithm) is what powers the modern world! It is used by:

  • Uber / Lyft: Matching available drivers to waiting riders.
  • Medical Boards: Matching graduating doctors to hospital residencies.
  • Dating Apps: Matching compatible users based on shared preferences.
