The Mathematics of Cartography

The Four Color Theorem

The Four Color Theorem states that any given contiguous map on a plane requires no more than four colors to color the regions such that no two adjacent regions share the same color.


This was the first major theorem to be proved using a computer in 1976 by Kenneth Appel and Wolfgang Haken. Our solver uses a Greedy Backtracking algorithm to search the possibility tree for a valid 4-coloring!
