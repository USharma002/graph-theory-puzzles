❓ Why does Grid Width matter? ▼
The "Taxicab" Jump:

1. Moving the empty tile Left/Right does not change the order of the numbers.

2. Moving it Up/Down makes it "jump" over (Width-1) numbers in the linear list.

• If Width is Odd (e.g. 3), it jumps 2 numbers. Jumping an even amount doesn't change Parity.
• If Width is Even (e.g. 4), it jumps 3 numbers. Jumping an odd amount flips Parity!

That's why Even width grids need to add the Row Distance to the calculation.
